# 2-5 ビルド/配布
- 開発: `pnpm dev` でWeb/PWA開発サーバーを起動。
- 本番ビルド: `pnpm build` でNext.jsをビルドし、PWAマニフェストとService Workerを出力。
- デスクトップ: Rust toolchainを準備し `pnpm tauri:build` でTauriバイナリを生成。データ保存先は各OSのappData配下。
- DB: スキーマ変更時は `pnpm db:push`、必要に応じて `pnpm seed` で初期データ投入。
- 配布: PWAはホスティング先にデプロイし、デスクトップは生成バイナリ/インストーラを配布。
