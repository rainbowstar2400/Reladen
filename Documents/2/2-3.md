# 2-3 主要技術
- フロント: Next.js(App Router)+TypeScript、Tailwind CSS、shadcn/uiでUIを構築。TanStack Queryでキャッシュ/フェッチ状態を管理し、Framer Motion等によるアニメーションを想定。
- データ: Drizzle ORMで型安全なスキーマを定義し、Supabase(Postgres)と連携。IndexedDB/Tauri KV(app.db)をローカル永続に利用。
- インフラ/周辺: Supabase Realtimeでライブ更新、next-pwaでService Worker/マニフェスト、Sentryで監視。デスクトップはTauri(Rust)でビルド。
- 開発環境: Node + pnpm。Rust toolchain(Tauri)が必要。ESLint/Prettier/Vitestを利用可能。
